### Q2：为什么选择反激拓扑而不是正激或者其他拓扑？

**回答：**

反激拓扑有三个核心优势让它成为这个项目的首选：

1. **电气隔离自然满足**：变压器原边和副边天然隔离，输入输出不需要额外隔离器件，安规容易过。
2. **多路输出扩展方便**：加辅助绕组即可提供多路输出（如本项目的 15V 辅助供电绕组）。
3. **拓扑极简**：只需一个开关管、一个二极管、一个电容，元件数量少，可靠性高。

**缺点**（也要承认）：
- 变压器利用率低，体积较大
- 纹波比正激大
- 只适合中小功率（本项目 60W 正好合适）

---

### Q3：这个项目用的是什么控制模式？为什么？

**回答：**

用的是 **峰值电流模式（Peak Current Mode）**，而不是电压模式。

**为什么选峰值电流模式：**

| 对比项 | 电压模式 | 峰值电流模式（本项目用） |
|--------|----------|------------------------|
| 响应速度 | 慢（二阶系统） | 快（一阶系统） |
| 输入扰动抑制 | 差 | 好（电流内环直接检测） |
| 过流保护 | 依赖额外电路 | 自然带过流保护 |
| 变压器饱和检测 | 无 | 有（电流检测） |

UC3842 本身就是一款经典的峰值电流模式控制芯片，内部自带电流比较器，通过检测原边电流峰值来控制 PWM 占空比。

## 二、反激拓扑原理类

### Q5：详细描述反激的工作原理，MOS导通和关断时能量怎么流动？

**回答：**

反激的本质是**能量耦合传递**——原边通磁、副边放能。

**MOS 导通阶段（t0~t1）：**
```
Vin → 变压器初级绕组 Np → 初级电流 Ip 线性上升
原边电感储能：W = ½·Lp·Ip²
此时副边二极管反向截止，副边无电流
输出电容 C 单独向负载供电
```

**MOS 关断阶段（t1~t2）：**
```
原边电流突然为0，变压器磁通不能突变
副边绕组感应出反向电压（上正下负）
二极管导通，副边电流 Is 线性下降
变压器储能通过二极管向输出侧释放
同时给输出电容补能
```

**关键波形要会说：**
- MOS 导通时：Vds≈0，Ip 上升（斜坡），Is=0
- MOS 关断时：Vds 升高到 Vin+反射电压，Ip=0，Is 下降

---

### Q6：什么是 CCM 和 DCM？这个项目是哪种？

**回答：**

- **CCM（Continuous Conduction Mode，连续导通模式）**：下一个周期到来时，副边电流还没降到零，变压器有残余能量。
- **DCM（Discontinuous Con导通 Mode，断续导通模式）**：副边电流在一个周期内降到零，变压器能量完全放完。

**本项目工作在 DCM 模式**，原因：
- 低压输入（110V）时，最大占空比约 48%，配合 KRP=0.8 的设计，副边电流在一个开关周期内可以降到零
- DCM 模式的特点是峰值电流大、但变压器磁芯复位简单、无右半平面零点（稳定性好）

---

### Q7：什么是反射电压（VOR）？它和最大占空比有什么关系？

**回答：**

反射电压 VOR 是 MOS 关断时，副边电压折算到原边的电压。

**计算过程：**
```
已知：Vbusmin = 110V，Vds(导通压降) = 4V，反射电压 VOR = 100V

Dmax = VOR / (VOR + Vbusmin - Vds)
     = 100 / (100 + 110 - 4)
     = 100 / 206
     ≈ 48.5%
```

**物理意义：** 占空比 D 本质上决定了原边通磁时间（储能）和关断放能时间的比例。Vbus 越低，为维持输出功率，通磁时间就要越长，所以 Dmax 出现在最低输入电压时。

---

### Q8：反激变压器和普通变压器有什么本质区别？

**回答：**

普通变压器：原边导通时副边同时有能量传递（共模）。

反激变压器：本质是一个**储能电感**，原边通磁储能，关断才向副边放能（原边和副边是**互斥**关系，不是同时）。

**等效理解：**
- 原边 = 电感线圈（储能阶段）
- 副边 = 从原边磁芯获取能量的感应绕组（放能阶段）

所以反激变压器设计时关注的是**电感量（储能能力）**，而不是像普通变压器那样关注耦合系数。

## 三、变压器设计类

### Q9：反激变压器是怎么设计的？一步一步说清楚。

**回答：**

**第一步：确定设计目标**
```
输出功率：Pout = 60W
输入电压：Vinmin = 110V，Vinmax = 375V
开关频率：fs = 100kHz
效率：η = 80%
脉动系数：KRP = 0.8（DCM模式）
```

**第二步：计算峰值电流**
```
Iavg = Pout / (Vinmin × η) = 60 / (110 × 0.8) = 0.682A
Ip = Iavg / [(1 - 0.5×KRP) × Dmax] = 2.341A
Ir = KRP × Ip = 1.873A
```

**第三步：计算初级电感量**
```
Lp = Pout × Ip² × KRP × (1 - 0.5×KRP) / [0.5 × (1-η) × fs × η]
   = 256.622μH
```

**第四步：选磁芯（AP值法）**
```
AP = (Lp × Ip² × 10⁴) / (Bw × Ko × Kj)^1.14 = 0.397cm⁴
选 PQ2625：AP = 0.997cm⁴ > 2×0.397 = 0.794 ✅
```

**第五步：计算匝数**
```
Ae = 118mm²（PQ2625）
Np = Vinmin × Dmax / (Ae × Bmax × fs)
   = 110 × 0.485 / (118×10⁻⁶ × 0.2 × 100×10³)
   = 30 匝

Ns = Np / Nps = 30 / 7.874 ≈ 4 匝
Nf = Ns × Vout1 / Vout = 4 × 15 / 12 = 5 匝（辅助绕组）
```

**第六步：线径选择（趋肤效应）**
```
趋肤深度：Dm = 2 × 66.1×10⁻³ / √fs = 0.418mm
→ 导线直径 < 0.418mm，可有效避免趋肤效应
初级：选 0.3mm × 3股（IP=2.34A）
次级：选 0.4mm × 11股（IsP=17.56A）
```

---

### Q10：什么是趋肤效应？选线时怎么考虑的？

**回答：**

**趋肤效应**：高频电流在导体中不是均匀分布，而是集中在导体表面（"皮肤"）流动，等效截面积减小，电阻增大。

**趋肤深度公式：**
```
Dm = 2 × 66.1 × 10⁻³ / √f  (mm)
   = 2 × 66.1×10⁻³ / √100000
   = 0.418mm
```

**设计结论：** 导线直径必须小于 0.418mm。本项目初级选 0.3mm，次级选 0.4mm，都满足要求。

**多股并绕的意义**：用多股细线代替单根粗线，增加总表面积，降低交流电阻。

---

### Q11：为什么要三明治绕法？原边和副边怎么排布的？

**回答：**

**三明治绕法**是变压器绕组的一种特殊排布方式，目的是**减少漏感**。

**本项目的绕组排布：**
```
第一层：初级 15匝（0.3mm×3股）—— 疏绕
第二层：辅助绕组 5匝 —— 疏绕（屏蔽层作用）
第三层：次级 4匝（0.4mm×11股）—— 疏绕（高压侧）
第四层：初级 15匝（0.3mm×3股）—— 疏绕
```

**物理理解：**
- 次级夹在两层初级中间 = 次级被初级"包裹"
- 这样初级和次级的磁耦合更好，漏磁减少
- 层间加了 2mm 以上挡墙（挡墙 = 绝缘挡块，防止高压侧对初级放电）

---

### Q12：气隙是怎么加的？加多少？

**回答：**

反激变压器**必须加气隙**，原因：防止磁芯饱和。

**原理：** 空气的磁导率远低于铁氧体，加入气隙后，相同安匝数下磁通密度更低，不容易饱和。

**本项目计算结果：**
- 气隙宽度 δ = 0.498mm（磨中柱方式）
- 加气隙后电感系数 ALG = 0.282μH/匝²（原始 AL=5250nH/匝²）
- 最终保证电感量 Lp = 256μH（1-3脚间，@100kHz测量）

## 四、功率器件选型类

### Q13：MOSFET 是怎么选型的？耐压和电流分别怎么算？

**回答：**

**第一步：算耐压**

MOS 关断时承受的电压平台为：
```
Vmos = Vinmax + Vbusmax × Np/Ns + VF
     = 375 + 375 × 7.874 + 0.7
     ≈ 3220V  （等等，这个不对...）

重新算：Vmos = Vinmax + VOR（反射电压）
     = 375 + 100 = 475V
```

加上漏感尖峰和 1.5 倍余量：
```
选型耐压 ≥ 475 × 1.5 = 712V
```

**实际选型：** IPA90R1K2C3
- 耐压 900V ✅
- 额定电流 5.1A ✅

---

### Q14：次级整流二极管怎么选？为什么要用肖特基？

**回答：**

**耐压计算：**
```
Vdio = Vout + Vinmax × Ns/Np
     = 12 + 375 × 4/30
     = 62V
留 1.5 倍余量：≥ 93V
```

**实际选型：** SBR20U100CT（肖特基二极管）
- 耐压 100V ✅
- 额定电流 10A ✅
- 正向压降 VF = 0.7V（肖特基比普通快恢复二极管更低，提升效率）

**为什么用肖特基：**
1. 正向压降低（0.3~0.7V），减少二极管损耗
2. 反向恢复时间极短（ns级），适合高频开关
3. 在低压输出场合（12V）优势明显

---

### Q15：输出电容怎么算的？为什么用 4 颗 470μF 并联？

**回答：**

**纹波要求：** ΔVout ≤ 100mV（设计目标）

**计算过程：**
```
Rout = Vout / Iout = 12 / 5 = 2.4Ω
Cout = Vout / (Rout × ΔVout × Dmax × fs)
     = 12 / (2.4 × 0.1 × 0.485 × 100×10³)
     = 103μF（理论值）
```

**实际选型考虑：**
1. **ESR 影响**：实际电容有等效串联电阻，纹波会比理论大 4 倍以上
2. **纹波电流**：次级峰值电流 17.5A，电容需能承受这个纹波电流
3. **选 4×470μF/35V 并联**：
   - 总容值 = 1880μF，远大于理论值，纹波更小
   - 多颗并联降低 ESR
   - 35V 耐压 > 12V×1.2倍安全系数 ✅

### Q16：RCD 钳位电路是干什么的？怎么设计？

**回答：**

**问题来源：**

变压器有漏感Lk（约 1%×Lp），漏感能量没有传递给副边，在 MOS 关断时会和 MOS 的 Coss（结电容）形成 LC 振荡，产生尖峰电压，可能损坏 MOS。

**RCD 钳位工作原理：**
```
MOS 关断 → 漏感能量给 Cc 充电 → 电容电压升高
 → 当电容电压 > Vclamp 时，二极管反偏截止
 → 电阻 Rc 消耗电容上的能量（缓慢放电）
 → 等待下一个周期重新吸收漏感能量
```

**本项目参数：**
```
漏感 Lk = 1% × 256μH = 2.566μH
钳位电压 Vclamp = 0.8 × 800V - Vbusmax = 265V
Rc = 64kΩ → 选 100kΩ
Cc = 312pF → 选 330pF/1kV（薄膜电容）
钳位二极管：FR107（1kV快恢复）
钳位电阻功率：1.129W → 选 2W 电阻（留余量）
```

## 五、控制环路类

### Q17：UC3842 的工作原理是什么？

**回答：**

UC3842 是**峰值电流模式**控制芯片，内部逻辑如下：

```
┌──────────────────────────────────────────┐
│  电压误差放大器（EA）                      │
│  FB脚 vs 内部2.5V基准 → 误差电压          │
│         ↓                                 │
│  电流比较器                               │
│  原边电流采样 vs 误差电压 → PWM关断信号     │
│         ↓                                 │
│  RS触发器                                 │
│  时钟上升沿→置位Q=1（MOS导通）            │
│  电流比较器翻高→复位Q=0（MOS关断）         │
│         ↓                                 │
│  图腾柱驱动（6脚输出）                    │
│  ±1A驱动能力                              │
└──────────────────────────────────────────┘
```

**关键管脚功能：**
| 管脚 | 名称 | 功能 |
|------|------|------|
| 1 | COMP | 电压环补偿端 |
| 2 | FB | 反馈电压输入（电压环负端） |
| 3 | ISENSE | 原边电流检测（电流环正端） |
| 4 | RT/CT | 开关频率设定（RC震荡） |
| 6 | OUTPUT | PWM驱动输出 |
| 7 | VCC | 芯片供电 |
| 8 | VREF | 5V参考输出 |

### Q18：为什么要用 TL431 + 光耦反馈？直接用电阻分压不行吗？

**回答：**

**核心问题：原边和副边需要电气隔离！**

```
反激原边：高压（~375V）
反激副边：低压（12V）
UC3842 在原边
输出电压在副边
 → 无法直接用电阻分压接到 FB 脚
```

**解决方案：TL431 + PC817 光耦**

```
光耦（PC817）= 隔离 + 信号传递
  原边发光二极管 ←→ 副边三极管（电气隔离）
  电流变化 → 光强变化 → 副边电流变化

TL431 = 2.5V 精密基准
  R脚电压 > 2.5V → K-A电流增加 → 光耦电流增加 → FB电压下降
  R脚电压 < 2.5V → K-A电流减小 → 光耦电流减小 → FB电压上升
```

**为什么光耦 CTR 不能直接用：**
- CTR 随温度漂移大，单独用光耦做线性反馈误差大
- TL431 内部有精密 2.5V 基准，用它作为比较基准可以提高精度

---

### Q19：环路补偿是怎么设计的？穿越频率和相位裕度是多少？

**回答：**

**穿越频率（带宽）设计：**
```
通常设为开关频率的 1/10（本项目 10kHz）
理由：大于1/10开关频率时，PWM采样延迟效应明显，系统不稳定
```

**补偿前系统特性：**
- 有两个穿越频率：500Hz 和 1.4MHz
- 高频段增益上翘 → 不稳定

**电压环补偿器设计：**
```
R9 = 1kΩ，C2 = 10nF，R17 = 1kΩ
传递函数：Gvp(s) = R9/R17 × (1 + sR9C2)
 → 零极点配置提升低频增益
```

**光耦+TL431 传递函数：**
```
R14 = 10kΩ，R10 = 20kΩ，R8 = 287Ω，C3 = 100nF
CTR = 0.8（PC817最小值）
```

**补偿后结果（通过 MATLAB Bode图验证）：**
| 指标 | 设计值 | 结果 |
|------|--------|------|
| 穿越频率 | 10kHz | 9.55kHz ✅ |
| 相位裕度 | ≥45° | 76.5° ✅ |
| 增益裕度 | ≥7dB | 14.2dB ✅ |
| 低频增益 | 高 | 40dB（100Hz）|

### Q20：什么是右半平面零点（RHPZ）？反激有没有？

**回答：**

**RHPZ 的本质：** 输出对占空比的响应有延迟——占空比增加，输出反而先降后升。

**反激 DCM 模式没有 RHPZ**（这是 DCM 的一个优势），但在 CCM 模式下存在。

| 工作模式 | 是否有 RHPZ | 原因 |
|----------|------------|------|
| DCM | 无 | 副边电流每个周期归零，不存在延迟 |
| CCM | 有 | 副边电流连续，占空比增加时，磁通建立需要时间，能量传递滞后 |

这也是为什么宽输入范围的电源通常设计在 DCM 模式——没有 RHPZ，环路补偿更简单。

## 六、硬件电路类

### Q21：这个项目的输入 EMI 电路是怎么设计的？

**回答：**

```
AC输入 → NTC(5D-11) → 保险丝(F1 2A) → 共模电感(L1 20mH)
                                              ↓
                                            L ──┬─ CY1 ──┬─ N
                                                              ← 共模滤波
                                               X ──┬─ CX1 ──┬─ N
                                                              ← 差模滤波
```

**各器件作用：**

| 器件 | 作用 |
|------|------|
| **NTC 5D-11** | 限流热敏电阻，抑制上电冲击电流（冷态电阻5Ω，上电后电阻降低） |
| **保险丝 2A** | 过流保护，故障时熔断 |
| **共模电感 L1** | 20mH，抑制共模干扰（LN线对地共模噪声） |
| **CY1/CY2** | 安规Y电容（1nF/250VAC），共模噪声对地通路 |
| **CX1/CX2** | 安规X电容（0.1μF/275VAC），差模滤波 |

### Q22：PCB 布局布线有哪些关键原则？反激的功率回路怎么布？

**回答：**

**本项目遵循的 PCB 布局原则：**

**1. 功能模块就近原则**
```
输入EMI → 整流桥 → 母线电容 → 反激原边 → 变压器原边侧
                                    ↓
                              变压器副边侧 → 输出整流 → 输出电容 → 负载
                                    ↓
                              光耦反馈 → UC3842
```
每个功能模块紧密排布，减少走线电感。

**2. 载流能力原则**
```
原边电流有效值 ~1A → 直流需0.4mm → 高频取5倍 → 实际用 2mm ✅
次级电流有效值 ~8A → 用铺铜方式走次级大电流网络 ✅
```

**3. 回路面积最小原则**

反激有**两个主要高频交流回路**：
- 回路1：变压器原边 → MOS → 初级电流采样电阻 → 母线电容负端
- 回路2：变压器副边 → 次级二极管 → 输出电容 → 变压器副边

两个回路面积都要尽可能小。

**4. 滤波电容就近原则**
```
输出电容 CE1/CE2/CE5/CE6
必须紧靠在输出整流二极管和输出端口之间
让输出电流先经过电容再流向负载
```

**5. 安规间距原则**

| 位置 | 电气间隙要求 |
|------|-------------|
| L-N（输入线间） | ≥2mm |
| 原边对副边（变压器两侧） | ≥4mm |
| 原边对大地 | ≥4mm |

### Q23：UC3842 的启动电路是怎么工作的？

**回答：**

**启动过程（先自举后自供电）：**
```
上电 → R12(10kΩ) 从母线取电 → VCC电容 CE4 充电
     → 当 VCC 达到 16V（启动阈值）→ UC3842 开始工作
     → 反激启动 → 辅助绕组建立电压
     → 辅助绕组 → 给 VCC 供电（自供电）
     → R12 退出工作
```

**关键点：**
- 启动电阻 R12 用 10kΩ（仿真用），实际可用 100kΩ（降低损耗）
- VCC 需要并联去耦电容 CE4（100μF/35V）稳定供电
- VCC 低于 10V 时芯片欠压保护关闭 PWM

### Q24：调试时你测了哪些波形？怎么判断电路是否正常？

**回答：**

**实测波形项目：**

| 测试项 | 正常波形特征 | 判断依据 |
|--------|-------------|----------|
| **输出电压纹波** | 峰峰值 ≤ 600mV（含噪声），开关纹波 ~20mV | 环路补偿到位 |

## 七、理论深度类

### Q25：反激开关频率选 100kHz 是怎么考虑的？能不能选更高或更低？

**回答：**

**选 100kHz 的权衡：**

| 频率升高 | 优点 | 缺点 |
|----------|------|------|
| | 变压器体积减小 | MOS开关损耗增加（开关次数多） |
| | 纹波减小 | 驱动损耗增加 |
| | 动态响应加快 | 辐射干扰增加（EMI更难处理）|

| 频率降低 | 优点 | 缺点 |
|----------|------|------|
| | 开关损耗低 | 变压器体积增大 |
| | EMI 容易处理 | 纹波增大 |
| | | 动态响应变慢 |

**本项目选 100kHz 的理由：**
1. UC3842 内部时钟振荡器易配置（8.5kΩ + 2.2nF）
2. 100kHz 是反激应用的主流频率，器件生态好
3. 变压器 PQ2625 在 100kHz 刚好能放下设计匝数

---

### Q26：峰值电流模式和电压模式在设计反激时有什么区别？

**回答：**

**峰值电流模式的优势（本项目用它的核心理由）：**

1. **固有的逐周期限流**：原边电流被限制，不易饱和
2. **输入电压扰动抑制好**：输入变化直接反映在电流斜坡上，响应快
3. **系统是一阶的**：传递函数比电压模式简单一个极点

**峰值电流模式的传递函数特点：**
```
Gvd(s) 有两个关键极点、一个ESR零点和
一个右半平面零点（在CCM时）
→ DCM模式无右半平面零点，更稳定
```

**电压模式的问题：**
- 需要大电感储能，动态响应慢
- 输入扰动需经过输出电容积分才反应出来

### Q27：设计报告中 bode 图是怎么画的？你会用 MATLAB 吗？

**回答：**

**会用。** 报告中的 bode 图用 MATLAB 绘制的，关键代码如下：

```matlab
Lp=256.622E-6; Vout=12; Pout=60;
Rload=Vout*Vout/Pout; Nps=30/4; Nps1=1/Nps;
D=0.485; Rsense=0.2; Esr=5E-3; Cout=1.88E-3;

wz=1/Esr/Cout;           % 电容ESR零点
wrz=Rload*(1-D)^2/Nps1^2/D/Lp;  % 右半平面零点(CCM时)
wp=(1+D)/Rload/Cout;    % 输出滤波器极点

Ak=1/3;  % UC3842内部1/3分压增益
s=tf('s');
Gvv=Ak*Rload*Nps*(1-D)*(1+s/wz)*(1-s/wrz)/(1+D)/Rsense/(1+s/wp);

margin(Gvv); grid on;
```

**Bode图关键参数解读：**
- **穿越频率**：9.55kHz（设计的10kHz ✅）
- **相位裕度**：76.5°（大于45° ✅，说明瞬态响应好且稳定）
- **增益裕度**：14.2dB（大于7dB ✅，说明相位接近0°时增益有余量）

## 八、综合扩展类

### Q28：如果让你把这个电源的输出功率从 60W 提高到 100W，你会修改哪些参数？

**回答：**

**需要重新评估的参数：**

| 参数 | 60W 现状 | 100W 需要调整 |
|------|---------|--------------|
| 变压器 | PQ2625 AP=0.997cm⁴ | 需要更大磁芯（如PQ3230或EI33） |
| 初级电感 Lp | 256μH | 可能需要减小（增大 Ip 来传递更多功率） |
| MOS | IPA90R1K2C3 5.1A | 需要更大电流（~8A），耐压不变 |
| 二极管 | SBR20U100CT 10A | 可能需要更大电流（~15A） |
| 输出电容 | 4×470μF | 增加容值或颗数，纹波电流更大 |
| RCD 钳位 | 100kΩ/330pF | 漏感能量更大，钳位电阻功率需增加 |
| 磁芯窗口填充 | KT=0.425 | 需要验证还能不能绕下 |

**最关键的改动是变压器**：需要重新按 AP 值法选更大的磁芯，重新计算所有参数。

### Q29：这个电源的效率 80% 是怎么保证的？损耗主要在哪里？

**回答：**

**效率分析：80% 意味着损耗 15W**

**主要损耗来源：**

| 损耗类型 | 来源 | 估算占比 |
|----------|------|---------|
| **MOS 开关损耗** | 每次开关的 Coss 充放电 | ~30% |
| **MOS 导通损耗** | IpRMS² × Rds(on) | ~20% |
| **二极管损耗** | 次级二极管导通压降 × 平均电流 | ~25% |
| **变压器铜损** | 趋肤效应导致交流电阻增大 | ~15% |
| **RCD 钳位损耗** | 漏感能量被电阻消耗 | ~8% |
| **其他（控制芯片、EMI等）** | | ~2% |

**提升效率的方向：**
- 选低 Rds(on) 的 MOS（如 CoolMOS）
- 用同步整流代替二极管（副边用 MOS 整流）
- 选 VF 更低的肖特基二极管

### Q30：这个项目有没有遇到什么问题？你是怎么解决的？

**回答方向（根据实际经历调整）：**

**可能遇到的问题及解决思路：**

**问题1：变压器绕制后电感量不对**
- 原因：气隙磨得不够精准，或绕线松紧不同
- 解决：重新磨中柱，用 LCR 表测量直到 Lp = 256μH

**问题2：输出纹波超标**
- 原因：输出电容 ESR 过大，或环路补偿参数不对
- 解决：增加输出电容颗数，或重新计算 R8/C3 参数

**问题3：MOS 尖峰电压过高**
- 原因：RCD 钳位参数不合适，或漏感比预期大
- 解决：增大钳位电容，或减小钳位电阻加快释放

**问题4：启动时输出电压超调**
- 原因：软启动参数或环路响应太快
- 解决：调整电压环补偿参数

### Q31：你认为这个项目里最难的设计点是什么？

**回答：**

我认为最核心也是最难的点是 **环路补偿设计**。

原因：
1. 涉及传递函数推导，需要扎实的自动控制理论
2. 光耦 CTR 的离散性（批次不同 CTR 不同）导致补偿参数要留足够余量
3. 理论计算和实际调试往往有差距，需要经验判断
4. 稳定性判据（相位裕度 45°、增益裕度 7dB）只是门槛，实际还需要调试

其他如器件选型、变压器设计都有比较成熟的计算公式可以参考，但环路补偿是理论+经验+调试的综合体现。

### Q32：你还做了 CBB02405D 双向 BUCK-BOOST 项目，反激和它有什么区别？

**回答：**

| 对比项 | CFY01205 反激 | CBB02405D 双向 BUCK-BOOST |
|--------|--------------|--------------------------|
| 拓扑 | 反激（隔离） | BUCK-BOOST（非隔离） |

## 十、个人理解类

### Q33：做完这个项目，你对开关电源设计有什么新的理解？

**回答方向：**

1. **理论计算只是起点**：设计报告算出来的参数只是理论值，实际调试会有偏差，需要反复迭代
2. **变压器是核心**：反激的几乎所有性能都由变压器决定——效率、纹波、发热、尺寸
3. **环路补偿是艺术也是科学**：既要有传递函数的理论支撑，也要有调试的经验积累
4. **EMI 问题不可忽视**：原理图没问题，PCB 布局不当会导致 EMI 测试失败
5. **电源设计是权衡的艺术**：效率 vs 成本 vs 体积 vs 可靠性，每个参数都是权衡的结果
